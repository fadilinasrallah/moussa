# LSB Steganography Detector

This project trains a binary detector for LSB steganography using the local
`ALASKA_v2_JPG_512_QF95_COLOR/` image folder as the clean cover distribution.
The app can:

- detect whether an image likely contains LSB steganography,
- embed UTF-8 text into a PNG using LSB,
- extract text from PNGs generated by this app.

## Dataset

The local dataset contains 20,000 color JPEG images at 512 x 512 and quality
factor 95. Public ALASKA2 material describes the full challenge as cover images
plus stego variants generated with JMiPOD, J-UNIWARD, and UERD. This repository
only has the processed cover folder, so the training script creates paired
stego PNGs with this project's LSB embedder.

References:

- ALASKA project page: https://alaska.utt.fr/
- Kaggle ALASKA2 competition: https://www.kaggle.com/competitions/alaska2-image-steganalysis/data

## Install

```bash
python -m pip install -r requirements.txt
```

## Train

Fast smoke test:

```bash
python run_pipeline.py --smoke --workers 8
```

Trained model used in the current workspace:

```bash
python run_pipeline.py --n 4000 --workers 15 --tune
```

Use all 20,000 covers when you want a longer full run:

```bash
python run_pipeline.py --workers 15 --tune
```

The pipeline:

1. samples ALASKA cover JPEGs,
2. exports clean control PNGs under `data/raw/clean_alaska_png/`,
3. generates matching LSB stego PNGs under `data/raw/stegano_alaska_lsb/`,
4. keeps each cover/stego pair in the same train/validation/test split,
5. extracts RGB texture and LSB features in parallel,
6. trains a calibrated linear SVM and a calibrated Random Forest,
7. tunes each model's decision threshold on the validation split,
8. writes models to `models/` and plots/metrics to `results/`.

Current tuned 4,000-cover run:

| Model | Accuracy | Precision | Recall | F1 | AUC |
| --- | ---: | ---: | ---: | ---: | ---: |
| Linear SVM | 0.9958 | 0.9934 | 0.9983 | 0.9958 | 0.9996 |
| Random Forest | 0.9983 | 0.9967 | 1.0000 | 0.9983 | 0.9993 |

## Detect From CLI

```bash
python detect.py --image ALASKA_v2_JPG_512_QF95_COLOR\00003.jpg --model rf
python detect.py --image some_image.png --model svm --json
```

## Run The App

```bash
streamlit run app.py
```

The app is designed around 512 x 512 images. For embedding, uploaded images are
resized to 512 x 512 first, then text is embedded, and the output is saved as
PNG so the LSBs are not destroyed by JPEG recompression.

## Important Limits

This is an LSB detector trained on app-generated LSB stego pairs. It is not a
general detector for JMiPOD, J-UNIWARD, or UERD unless you add those stego
classes to the training data. Very small payloads are detected reliably for
images generated by this app because the model sees the app's 32-bit LSB length
header.
